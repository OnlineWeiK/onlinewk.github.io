2012年暑期小记

发表于 2012-07-18   |   分类于 随笔

现在每天的生活感觉很充实，终于有时间来做自己喜欢的和一直想做的事情，背背单词，看看自己喜欢的小说，在图书馆待上一整天，偶尔发发呆，听听歌。中午水饭足饭（木有酒）后看看宫崎骏的动画，听听幸福课，累的时候趴下休息，渴的时候喝点kate给的菊花茶而且还加冰糖，偶尔呢，看看美女帅哥们奋斗的样子。有时不定时的能感觉到Hale睡觉时脚突然一蹬，然后我旁边睡觉的人突然的就醒了（一蹬惊醒梦中人啊），然后我用眼睛的余光感觉到她用余光瞥了Hale一眼，看他睡得正香，她一脸无奈！发现每到要吃饭的时间周围变得热闹了，我想大家终于可以放下紧绷的神经和同学聊聊天，谈谈心了。然后是晚上的时间，基本是两三天就会去运动运动，特别是跑步的时候，当你刚跑两三圈时会感觉脚开始酸了，但其实每当这时候你要告诉自己再咬咬牙，再坚持会，你就会发现越过一个一个的临界点，就算让你再跑几圈也没问题！想想那些在图书馆奋斗的学长学姐们，他们也要经历这样一个个的临界点，每当累了，想放弃时，他们心里的另一个声音一直在告诉自己：“咬咬牙，再坚持会！Yes.You can make it！”天降大任于斯人也，必先苦其心志，劳其筋骨…跑完步后躺在草坪上看着星星，想着想着发现这天空还真是圆的，如果古人那时也躺着瞅瞅这星空是不是就会有天圆地也圆的说法了。然后是今晚特别想吃卤面，因为有点晚了，想着如果学生街那边的门关了，为了卤面我就再翻次墙，但是，当我到紧闭的门前时发现这会翻墙也没用了，人家收摊了。然后一个人回来去兵临城下买了杯爱喝的草莓奶茶加子爪子，刚好碰到汉群就和他在那边聊着奶茶店里那一本一本的征男友女友的厚本子，有个女的还写着有电摩者优先，非诚勿扰~记得上次无聊在那把一个征男友的号码改成了1008610000。然后十点半回宿舍，发现，丫的，楼下的门是坏的，一个想法突然闪过：要是半夜有人拿着杀猪刀冲到宿舍，我是该用电脑挡呢还是用书挡呢，还是说：“嗨！哥们，刀不错么，哪里买的？”…好像挺久没运动了…回到宿舍后就开始洗澡了，愣是在厕所边洗边唱折腾了半个小时，单曲重唱~，然后算洗完衣服后写了以上839个字，该睡觉了，晚安！

Petya and Java
